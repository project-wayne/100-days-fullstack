💡 Key Takeaways
HTML = structure (what content is)
CSS = styling (how it looks)
<head> = page info (not visible)
<body> = visible content
Good practice = separate structure and styling

⚠️ Mistakes I learned from

Understanding that not all CSS properties inherit
Learning proper placement of <link> inside <head>
